# SSGI Front-End

A modern front-end application for the SSGI project, built with Next.js, Tailwind CSS, and TypeScript.

## Features

- **Next.js App Router**: Optimized for performance and SEO.
- **Tailwind CSS**: Utility-first styling for rapid development.
- **TypeScript**: Type-safe code for better developer experience.
- **Responsive Design**: Works across mobile, tablet, and desktop devices.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/`: Contains the application routes and layouts.
- `components/`: Reusable UI components.
- `hooks/`: Custom React hooks.
- `lib/`: Utility functions and shared logic.
- `public/`: Static assets like images and fonts.
- `styles/`: Global CSS and Tailwind configuration.

## Environment Variables

Make sure to create a `.env.local` file in the root directory and add any necessary environment variables. See `.env.example` (if available) for reference.
